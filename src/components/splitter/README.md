# Splitter

Create vertically and horizontally moveable splitter bar.

## Examples

```js
class Example extends React.Component {
  render() {
    return(
        <Splitter>
          <div>1</div>
          <div>2</div>
          <div>3</div>
          <div>4</div>
        </Splitter>
    )
  }
}
```
